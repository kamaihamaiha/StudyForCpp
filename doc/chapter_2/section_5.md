## 处理类型

- 类型别名
- auto 类型说明符
- decltype 类型指示符

### 类型别名(type alias)

使用关键字：**typedef**
```c++
typedef double wages;     // wages 是 double 的同义词
typedef wages base, *p;   // base 是 double 的同义词，p 是 double* d的同义词
```
